# Virtual Display Driver API

A project for interfacing with the [Virtual Display Driver (VDD)](https://github.com/VirtualDrivers/Virtual-Display-Driver) — a Windows Indirect Display Driver that creates virtual monitors without physical hardware.

## Documentation

- [VDD Overview](docs/VDD-Overview.md) — What VDD is, key capabilities, architecture, and configuration
- [Pipe API Reference](docs/pipe-api-reference.md) — API reference for the named-pipe control protocol (commands, responses, quick-start examples)
- [Pipe Technical Guide](docs/pipe-technical-guide.md) — Deep dive into the wire protocol, connection model, encoding, and implementation details
