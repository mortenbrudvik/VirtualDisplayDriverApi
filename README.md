# Virtual Display Driver API

A project for interfacing with the [Virtual Display Driver (VDD)](https://github.com/VirtualDrivers/Virtual-Display-Driver) — a Windows Indirect Display Driver that creates virtual monitors without physical hardware.

- [VDD Overview](docs/virtual-display-driver-overview.md) — What VDD is, key capabilities, architecture, and configuration
- [Named Pipe API Reference](docs/named-pipe-api-reference.md) — API reference for the named-pipe control protocol (commands, responses, quick-start examples)
- [Named Pipe Technical Deep Dive](docs/named-pipe-technical-deep-dive.md) — Deep dive into the wire protocol, connection model, encoding, and implementation details
